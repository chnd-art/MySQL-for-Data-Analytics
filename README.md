# 🗄️ MySQL for Data Analytics

This repository contains data analysis projects using SQL, focusing on real-world datasets and business insights.

## 📁 Projects

* 👶 **Baby Name Trend Analysis**
* 🍽️ **Restaurant Order Analysis**

Each project explores data using SQL queries, aggregations, and analytical techniques to uncover meaningful insights.

* SQL (MySQL)
* Data Exploration
* Data Cleaning
* Joins (INNER JOIN, LEFT JOIN)
* Aggregations (SUM, COUNT, AVG)

* Subqueries & CTEs
* Business Data Analysis

## 🚀 Purpose

The goal of this repository is to demonstrate how SQL can be used to:

* Analyze structured data efficiently
* Extract actionable business insights
* Build a strong foundation in data analytics
